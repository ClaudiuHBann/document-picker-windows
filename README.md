# @hbannro/rn-documents-picker-windows

Pick documents on windows.

## Installation

```sh
npm install @hbannro/rn-documents-picker-windows
```

## Compatibility

| @hbannro/rn-documents-picker-windows | @react-native-documents/picker |
| ------------------------------------ | ------------------------------ |
| 10.0.1                               | `=10.0.1`                      |
| 10.1.0                               | `=10.1.0`                      |
| 10.1.1                               | `>=10.1.1 <10.1.4`             |
